My projects

Decentralized Group Chat

Created a decentralized group chat application with a team of three using ReactJs, Tailwind, MongoDB, and IndexedDB with CRDTs for synchronization and WebRTC for communication This was the final year project work done for my Bachelor's degree.

Credit Score: Service Learning

Developed a web application (CreditScore) with a team of five to allot extra credits to students for volunteering activities of CAPS (Center for Academic and Professional Support) as part of our Service Learning Project My main responsibilities were handling the front-end, group meetings and modeling the application.

Personal Project : Stock Analysis App

Created a web application using ReactJS with Bootstrap for the front end and Python Flask for the back end. Used an online API service to fetch real-time stock data and then performed moving average analysis on the stock data.

ERP Course Plan : IIIC Internship

Made dynamic webpages using Angular and Bootstrap for Student Course Plan(ERP). My contribution was creating CRUD operations for entering course details and scheduling and editing meetings.

My Opinion

I recently watched AI for Education by Sal Khan, a TED talk from last year, in which he discussed how AI can transform the world by providing extra support to students through Khan Academy's new AI-powered bot, Khanmigo. This smart tutor can help with problems across various subjects and contexts with great ease. It also includes a teacher version, where educators can access explanations, generate reports, and plan lessons. This allows teachers to spend more time with students one-on-one. AI can be a boon or a bane; it depends on how we use it and ensure it remains secure so that it doesn't fall into the wrong hands.
